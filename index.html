Etiquetas HTML

Elementos raíz

Las etiquetas <!doctype html> y <html> se ubican al inicio de nuestra página web e indican que el documento está bajo el estandar HTML5 y le indican al explorador que lo que viene a continuación debe ser interpretado como código HTML.

Metadatos del documento

Estas etiquetas se ubican en la abecera del documento HTML. Representa una colección de metadatos acerca del documento, incluyendo enlaces, hojas de estilo titulo del documento e información relacionada.

Las podemos ver como: <head>, <title>, <base>, <meta>, <link>, <style> entre otras.

<h2>Secciones</h2>

Son las etiquetas que encontramos en el <body> de la página (Sí, el body también es una etiqueta de sección 😉), estas pueden ser: <section>, <nav>, <article>, <aside>, <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <header>, <footer>, <main> entre otras.

Agrupación de Contenido

<p>Como su nombre lo indica, estas etiquetas agrupan el contenido en diferentes formatos que podemos usar según el contexto de lo que queremos indicar. Por ejemplo este párrafo 😉!</p>

Podemos utilizar la etiqueta <hr> para incluir una separación entre parrafos o secciones ↓

<blockquote>Esta etiqueta podemos usarla cuando queremos citar un contenido de otra fuente y podemos darle estilos diferentes al resto del contenido.</blockquote>

Si por otro lado queremos mostrar información en forma de listas, podemos utilizar las etiquetas <ul> para listas desordenadas (Unordered List) y/o la etiqueta <ol> para listas ordenadas (Ordered List). Semántica a nivel de texto

Estas etiquetas nos permiten destacar elementos de texto dentro de un mismo párrafo, podemos utilizar la etiqueta <strong>para destacar</strong> alguna palabra o frase. Podemos agregar referencias <sub>(1)</sub> en el texto y luego explicarlas <sup>(2)</sup> en otro apartado utilizando la etiqueta <small>, por ejemplo:

<small>(1)Esto es una referencia y (2)Esto también.</small>

Si quieres agregar un enlace puedes usar la etiqueta <a> con sus atributos.

  • Si quieres abrir este enlace en una nueva pestaña, haz click aquí!

    • <a href="https://www.youtube.com/" target="_blank">

  • Si quieres abrir este enlace en esta pestaña, haz click aquí!

    • <a href="https://www.youtube.com/" target="_self">

Para agregar un espacio entre un párrafo y otro (Como cuando usamos enter en un documento de word) utilizamos la etiqueta <br>


El espacio entre el párrafo anterior y este es más amplio. Si queremos destacar alguna palabra o frase en el texto pero de otra forma que no sea en <i>italic</i> o <strong>strong</strong>, podemos utilizar la etiqueta <span> y combinarla con atributos que nos permitirán mostrar el contenido de forma diferente al resto del texto.
